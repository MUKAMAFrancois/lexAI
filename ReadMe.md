- Enterprise Contract Intelligence Agent
This is an AI-powered compliance agent that automates the review of B2B contracts, extracting critical data and flagging risks against company policy in seconds.

# 1. Setup

- install: `npx create-next-app@latest lexai-frontend`
- libraries: `npm install axios lucide-react clsx tailwind-merge`, We need `axios` for API calls and `lucide-react` for those professional dashboard icons (Shields, Warnings, Checks).

# 2. Run

command: `npm run dev`
